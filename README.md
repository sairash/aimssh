## SSH Pomo
This is a terminal pomodoro application.

To use the application:
``` sh
ssh pomo.sairashgautam.com.np
```

Or use it directly

``` sh
git clone https://github.com/sairash/pomossh
```

``` sh
cd pomossh
```

``` sh
go build
```

``` sh
./pomossh
```
